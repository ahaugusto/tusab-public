Brain'IAC — TERMOS DE USO E INFORMAÇÕES DE INSTALAÇÃO
============================================================

DOWNLOADS ADICIONAIS NECESSÁRIOS
----------------------------------
O Brain'IAC requer componentes adicionais que serão baixados
automaticamente na primeira execução:

  • Ollama (~50 MB)
    Motor de inteligência artificial local. Permite usar o Agente IA
    sem custos e sem conexão com servidores externos.

  • Modelo de linguagem llama3.2:1b (~1,3 GB)
    Baixado apenas na primeira vez que o Agente IA for utilizado.
    O download é exibido com barra de progresso dentro do aplicativo.

Uma conexão com a internet é necessária apenas para esses downloads
iniciais. Após a configuração, o Brain'IAC funciona completamente offline.

3. INTEGRAÇÃO COM GOOGLE DRIVE
   O acesso ao Google Drive é limitado a arquivos criados pelo próprio
   aplicativo (escopo drive.file). Nenhum outro arquivo é acessado.

4. CHAVES DE API
   Chaves de API inseridas pelo usuário são armazenadas localmente e
