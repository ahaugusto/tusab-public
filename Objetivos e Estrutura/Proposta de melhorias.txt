─────────────────────────────────────────
IMPLEMENTADO
─────────────────────────────────────────

EXTRAÇÃO
✅ Seletor de tipos de conteúdo (Vídeos / Shorts / Lives / Podcasts / Cursos / Playlists)
✅ Sync incremental com Drive (sobe a cada parte completada)
✅ Delta inteligente — não reextrai vídeos já processados
✅ Coleta de hashtags/tags do YouTube via yt-dlp (campo TAGS no .txt)
✅ Coleta de descrição dos vídeos (campo DESCRICAO no .txt)
✅ Metadados do canal: nome, handle, inscritos, data de extração
✅ Cabeçalho de atribuição em cada arquivo .txt gerado

AGENTE RAG
✅ Substituição de ChromaDB + embeddings por BM25 local (sem API externa)
✅ Indexação instantânea — leitura de .txt + JSON local, sem chamadas de rede
✅ Cache BM25 em memória — índice carregado uma vez, latência <100ms
✅ Enriquecimento de corpus: tags (3×) + keywords TF-IDF (2×) + descrições
✅ Query expansion — LLM gera 2 variações da pergunta, scores por média
✅ Histórico de conversa — últimas 6 trocas no contexto do LLM
✅ Streaming de respostas — texto palavra por palavra, cursor piscante
✅ Multi-canal — indexar N canais, buscar em todos simultaneamente
✅ Deletar índice individual por canal
✅ Suporte a Ollama local (llama3.2:1b) como provedor padrão gratuito

INTERFACE
✅ Accordions na aba Agente (Provedor de IA + Base de Conhecimento)
✅ Hint de primeiro acesso à aba Agente
✅ Card de canal indexado com nome, handle e inscritos
✅ Botões de pasta distintos (cerebro_txt × gestao)
✅ Tema automático via prefers-color-scheme com persistência
✅ Resumo pós-extração com modal de próximos passos
✅ Onboarding de 5 passos na primeira abertura
✅ Log em tempo real com altura adaptativa (clamp responsivo)
✅ Responsividade via clamp() em vez de breakpoints fixos

DISTRIBUIÇÃO
✅ Electron 34 com Python embeddable bundled
✅ Auto-update via electron-updater + GitHub Releases
✅ Ollama baixado automaticamente na primeira execução (Win/Mac/Linux)
✅ Detecção de OS para download correto do Ollama
✅ Loading screen com status em tempo real durante setup do Ollama
✅ Aviso de downloads adicionais no instalador NSIS
