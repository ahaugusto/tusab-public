a. Exibe a data e a hora em que o trâmite foi realizado;
3. Tipo do trâmite;
a. Exibe o tipo de trâmite realizado (Vide tabela dos tipos de trâmites realizados no
Plataforma Brasil);
4. Versão;
a. Exibe a versão do Protocolo (Projeto/Emenda) que está sendo detalhado;
5. Autor;
a. Para o módulo Pesquisador, esta coluna não é exibida;
b. Para os módulos CEP, CONEP e Administrador, esta coluna deve exibir o nome
social do usuário que realizou a ação do trâmite;
i. Nos módulos CEP e CONEP, a coluna Autor exibirá informação apenas
para os trâmites realizados pelo próprio Centro e pelo Pesquisador. Para
os demais trâmites, o sistema exibe uma imagem de “cadeado” na coluna
Autor;
ii. Para o módulo Administrador, a coluna Autor sempre exibirá informação;
6. Perfil
a. Esta coluna deve exibir o perfil do usuário em que gerou o trâmite.
i. Caso o usuário que tenha realizado o trâmite tenha o perfil de
Administrador, é este perfil que deve ser registrado no histórico de trâmites,
independente deste usuário ter algum outro perfil atribuído a ele.
ii. Caso o usuário tenha realizado o trâmite tenha o perfil de Coordenador (e
não for Administrador), é este perfil que deve ser registrado no histórico de
trâmites, independente deste usuário ter algum outro perfil atribuído a ele.
iii. Se o usuário que tenha realizado o trâmite não tiver perfil de Administrador
e nem perfil de Coordenador, o sistema deve registrar e exibir o perfil
logado no momento da realização deste trâmite.
7. Origem;
a. Exibe a origem da ação do referido trâmite;
Se a ação foi gerada pelo Pesquisador, deve armazenar e exibir o texto
“Pesquisador”;
Se a ação foi gerada pelo CEP responsável, deve armazenar e exibir o nome do
CEP;
Se a ação foi gerada pela CONEP, deve armazenar e exibir o texto “CONEP”.
