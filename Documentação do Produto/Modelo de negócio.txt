CATEGORIA
─────────────────────────────────────────

ISV — Independent Software Vendor
Modelo: Freemium + Perpetual Seat License + Subscription (B2B)

O Brain'IAC é um software desktop vendido por licença.
Não há servidor central. Não há assinatura obrigatória.
O valor está no software instalado na máquina do cliente.

─────────────────────────────────────────
O QUE É UMA LICENÇA DE SOFTWARE DESKTOP
─────────────────────────────────────────

Perpetual License
  → Compra única. O cliente é dono da versão adquirida.
  → Atualizações de patch (0.x.X) sempre gratuitas.
  → Atualizações de minor/major (0.X.0 / X.0.0) podem ser pagas.
  → Modelo clássico: Adobe (antes do Creative Cloud), Office perpétuo.

Seat (cadeira)
  → Uma licença = uma instalação autorizada = um usuário.
  → O sistema de licença valida: chave + hardware fingerprint.
  → Se o usuário mudar de máquina, pode transferir a licença.

TIERS DE PRODUTO
─────────────────────────────────────────

FREE — Gratuito
  Público: qualquer usuário individual
  Inclui:
    · Extração ilimitada de qualquer canal YouTube
    · Agente RAG local com Ollama (llama3.2:1b)

    · Recursos avançados dos tiers pagos

PRO — Licença Perpétua (1 seat)
  Público: profissional, pesquisador, criador individual
  Modelo: compra única (sugestão: R$97–R$197)
  Inclui tudo do Free +:
    · Suporte por email
    · Multi-canal simultâneo ilimitado
    · Exportação do histórico de chat (PDF/TXT)
    · Acesso antecipado a novas features


STUDIO — Assinatura (criador com audiência)
  Público: criador de conteúdo com canal próprio e base de alunos
  Modelo: mensalidade (sugestão: R$97–R$297/mês)
  Inclui tudo do Pro +:
    · Chatbot hospedado acessível para alunos via link
    · Painel de analytics: perguntas mais frequentes, temas buscados
    · Customização de marca (logo, nome do assistente)
    · Atualização automática da base quando novo vídeo é publicado


ENTERPRISE — Contrato Anual
  Público: instituições (universidades, cursinhos, empresas)
  Modelo: licenciamento por volume + contrato (sugestão: sob consulta)
  Inclui tudo do Studio +:
    · N seats configuráveis
    · Onboarding e implementação assistida
    · SLA de suporte (resposta em 24h)
    · Possibilidade de deployment on-premise
    · NDA e contrato personalizado

─────────────────────────────────────────
INFRAESTRUTURA DE LICENÇA
─────────────────────────────────────────

Plataforma escolhida: Lemon Squeezy
  → Gerencia: pagamento, chave de ativação, webhooks, reembolsos
  → Integra com Electron via validação HTTP simples
  → Suporta: one-time, assinatura, seat-based
  → Emite nota fiscal automática para pessoa jurídica

Fluxo de ativação:
  1. Usuário compra no Lemon Squeezy → recebe chave por email
  2. Abre o Brain'IAC → tela de ativação na primeira abertura
  3. Cola a chave → app valida com API do Lemon Squeezy
  4. Licença vinculada ao hardware (fingerprint) → ativa

Pré-requisito técnico: implementar tela de ativação no Electron

PROTEÇÃO DO PRODUTO
─────────────────────────────────────────

· Software rodando localmente = código pode ser extraído
· Mitigação atual: copyright + INPI + headers em todos os arquivos
· Mitigação futura: PyArmor ou Nuitka para compilar o backend Python
· A licença de uso (EULA) proíbe redistribuição e engenharia reversa
· O modelo Free legítimo reduz incentivo de piratear a versão paga
