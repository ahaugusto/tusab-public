IMPLEMENTADO
─────────────────────────────────────────

EXTRAÇÃO
✅ Seletor de tipos de conteúdo (Vídeos / Shorts / Lives / Podcasts / Cursos / Playlists)
✅ Sync incremental com Drive (sobe a cada parte completada)
✅ Delta inteligente — não reextrai vídeos já processados
✅ Coleta de hashtags/tags do YouTube via yt-dlp (campo TAGS no .txt)
✅ Coleta de descrição dos vídeos (campo DESCRICAO no .txt)
✅ Metadados do canal: nome, handle, inscritos, data de extração
✅ Cabeçalho de atribuição em cada arquivo .txt gerado
✅ Notificação nativa do OS ao concluir extração

BASE DE CONHECIMENTO
✅ Multi-fonte: YouTube + PDF + DOCX + MD + TXT + texto colado
✅ Estrutura organizada: cerebro/youtube/, documentos/, textos/
✅ Migração automática de cerebro_txt/ → cerebro/youtube/
✅ Gestão completa via interface (adicionar, visualizar, deletar)
✅ BM25 indexa todas as fontes simultaneamente

AGENTE RAG
✅ BM25 local (sem API externa, sem embeddings)
✅ Cache BM25 em memória — latência <100ms
✅ Enriquecimento: tags (3×) + keywords TF-IDF (2×) + descrições
✅ Query expansion — 2 variações da pergunta, scores combinados
✅ Histórico de conversa — últimas 6 trocas no contexto
✅ Streaming de respostas com cursor piscante
✅ Multi-canal — busca simultânea em N canais
✅ Anti-alucinação: threshold BM25 + verificação pós-geração
✅ Ollama local (llama3.2:1b) como provedor padrão gratuito
✅ Groq: llama-3.1-8b (rápido) e llama-3.1-70b (qualidade) — gratuitos
✅ Chat flutuante (drawer lateral) acessível de qualquer aba
✅ Chat vinculado ao canal configurado no sidebar
